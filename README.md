## Description

in this repo I've refactored the code in order using the postgres to store and manipulate data, instread of the
in-memory array.

## Installation

```bash
$ yarn install
```

## Running the app

```bash
# development
$ yarn run start

# watch mode
$ yarn run start:dev

# production mode
$ yarn run start:prod
```
